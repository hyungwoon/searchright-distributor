## 핵심 서비스

### 한 줄 정의
**역량 메타데이터 기반 AI 인재 발굴 솔루션** — 채용 공고로 만날 수 없는 소극적 구직자(66%)를 AI로 발굴하여 커피챗까지 연결

### 3대 핵심 기능

1. **역량 메타데이터 증강**: 다양한 플랫폼의 인재 경력 데이터를 AI로 증강하여 기존에 검색되지 않던 인재 발굴
2. **핵심 인재 리스트 제공**: 포지션별 최적합 후보자 리스트 + 적합 사유/인사이트/추천 순위 제공
3. **AI 초개인화 메시지 생성**: 인재별 맞춤 메시지로 커피챗 전환율 극대화

### 기술 스택 및 방향

- **현재**: 리스트 제공형 서비스 모델
- **전환 중**: SaaS 플랫폼으로의 전환 추진
- **AI 기술**: LLM 기반 LangGraph 활용, RAG(Retrieval-Augmented Generation) 구조로 후보자 데이터 자동 수집·가공·추천 시스템 구축
- **데이터 소스**: 리멤버, 링크드인, 보도자료 등 공개/비공개 외부 데이터 기반 역량 정보 추출

### 차별점 (경쟁 우위)
- **독자적 역량 정보 추출** → 기존 플랫폼에서 검색 불가능한 인재 발굴
- **데이터 증강** → 부족한 인재 정보를 메타데이터 기반으로 보강
- **개인화 메시지** → 콜드 메시지의 회신율/전환율 극대화
- **비용 효율** → 헤드헌팅 대비 1/3 수준의 수수료율 (7% vs 통상 20~25%)

4. **비용 압박**: 전통 헤드헌팅(20~25%) 대비 효율적 대안 수요 → AI 기반 소싱 플랫폼 성장

### 경쟁 우위 분석

#### 서치라이트AI의 강점 (vs 경쟁사)
1. **역량 메타데이터 증강**: 경쟁사가 보유 DB 내 검색에 그치는 반면, 서치라이트AI는 외부 데이터를 증강하여 "검색되지 않던 인재"를 발굴 — 이것이 가장 핵심적 차별점
2. **초개인화 메시지**: 단순 추천을 넘어 인재별 맞춤 콜드 메시지 생성 → 회신율 40% (업계 3배)
3. **비용 효율성**: 헤드헌팅 대비 1/3 수준 수수료, 리스트형은 100만원 고정
4. **시니어/희소 포지션 특화**: 범용 플랫폼이 다루기 어려운 C-level, VP, 테크리드 등에 집중

#### 서치라이트AI의 과제 (vs 경쟁사)
1. **인재풀 규모**: 자체 인재풀이 아닌 외부 데이터 의존 → 데이터 접근 안정성 리스크
2. **브랜드 인지도**: 2024년 설립 초기 스타트업 → 원티드·리멤버 대비 인지도 열세
3. **SaaS 전환**: 현재 컨설팅+리스트형 → SaaS 전환 과정에서 운영 효율·확장성 과제
4. **스케일러빌리티**: 인력 기반 서비스 요소가 남아 있어 고객 확장 시 병목 가능성

## SWOT 분석

### Strengths (강점)
- 역량 메타데이터 증강이라는 명확한 기술 차별점
- 높은 성과 지표 (회신율 40%, 업계 3배)
- 시니어·희소 포지션이라는 고부가가치 니치 점유
- TIPS 선정으로 기술력·성장성 검증
- 창업팀의 채용 실무 경험 (CPO, 사업개발 출신)
- LangGraph + RAG 기반 자체 AI 기술 개발

### Weaknesses (약점)
- 2024년 설립 초기 스타트업으로 낮은 브랜드 인지도
- 자체 인재 DB 부재 → 외부 데이터 소스 의존
- 현재 컨설팅+리스트형 모델의 확장성 한계
- 소규모 팀으로 대량 고객 대응 역량 제한
- SaaS 전환 과도기 → 제품 완성도 리스크

### Opportunities (기회)
- 다이렉트 소싱 비율 51.2%로 시장 주류화
- 경험 중심 채용 트렌드 (역량 메타데이터 수요 증가)
- HR Tech 시장 CAGR 7.55% 안정 성장
- 글로벌 HR용 AI 시장 CAGR 24.4% 고성장
- 4~7년차 "중니어" 채용 수요 집중 (58.9%)
- 전통 헤드헌팅의 비용 비효율에 대한 기업 불만
- SaaS 전환 성공 시 매출 스케일 확보

### Threats (위협)
- 원티드·리멤버 등 기존 플랫폼의 AI 소싱 기능 강화
- 잡코리아(2,400만 인재풀) 등 대형 플랫폼의 자원 우위
- 글로벌 AI 소싱 플랫폼의 한국 시장 진출 가능성
- 데이터 규제 강화 시 외부 데이터 수집 제한 리스크
- AI 기술 평준화로 기술 차별점 희석 가능성
- 경기 침체 시 채용 예산 축소 → 유료 솔루션 도입 지연

## 전략적 시사점

### 핵심 포지셔닝 권고
서치라이트AI는 **"AI 역량 메타데이터 기반 시니어 인재 발굴 전문"**이라는 니치 포지셔닝을 강화해야 한다. 범용 채용 플랫폼과의 정면 경쟁은 인재풀·브랜드 격차로 불리하며, "검색되지 않는 인재를 찾는다"는 고유 가치를 극대화하는 것이 경쟁 우위의 핵심이다.

### 우선 과제
1. **SaaS 전환 가속**: 컨설팅 의존도를 줄이고 셀프서브 플랫폼으로 전환 → 확장성 확보
2. **성과 데이터 축적·공개**: 회신율 40%, 커피챗 성사율 등 정량 성과를 케이스 스터디로 체계화 → 영업·마케팅 무기
3. **데이터 소스 다변화**: 특정 플랫폼 의존 리스크를 줄이기 위해 데이터 소스 확장
4. **인지도 확대**: 블로그 콘텐츠 마케팅 + 고객 사례 공개로 브랜드 구축
